# 🛠️ 工具箱 (Toolbox)

本目錄包含用於影片處理、分析和管理的各種實用工具腳本。

## 📋 工具列表

### 1. 影片分割與剪輯
- **`video_splitter_gui.py`**
  - **用途**: 全功能的圖形化影片分割工具。
  - **功能**: 
    - 視覺化時間軸操作
    - 支援多區間選擇與批次分割
    - 整合 FFmpeg GPU 硬體加速
  - **使用方式**: 直接執行腳本啟動 GUI 介面。

- **`video_clip.py`**
  - **用途**: 簡單的命令列影片剪輯工具。
  - **功能**: 指定起始與結束幀數來剪裁影片，支援調整播放速度。

### 2. 影片同步與元數據
- **`video_metadata_sync.py`**
  - **用途**: 雙鏡頭影片同步分析。
  - **功能**: 讀取影片的時間碼 (Timecode) 元數據，計算兩部影片的同步點和相對幀數。

- **`video_metadata.py`**
  - **用途**: 讀取單一影片的元數據。
  - **功能**: 提取影片的起始時間碼、結束時間碼和總幀數。

- **`video_clip_sync.py`**
  - **用途**: 影片同步剪輯處理。
  - **功能**: 根據擊球幀 (hit frame) 同步兩部影片，並進行裁切輸出。

### 3. 播放與截圖
- **`video_player.py`**
  - **用途**: 簡易影片播放器。
  - **功能**: 支援播放/暫停、調整播放速度 (加速/減速) 的 OpenCV 播放器。

- **`video_grab_photo.py`**
  - **用途**: 雙鏡頭同步截圖工具。
  - **功能**: 同時播放兩部影片，支援逐幀控制 (前進/後退)，並可隨時截取畫面保存。

### 4. 檔案管理與工具
- **`rename.py`**
  - **用途**: 批量檔案重命名。
  - **功能**: 針對特定目錄結構 (如 `junior/`) 下的 45度角與側面影片進行標準化重命名。

- **`video_writer_utils.py`**
  - **用途**: 影片寫入通用模組。
  - **功能**: 提供 `safe_video_writer` 函數，自動嘗試多種編碼器 (mp4v, H264, XVID 等)，解決不同系統下的影片儲存相容性問題。

## 🚀 使用說明

大部分工具可直接透過 Python 執行：

```bash
# 啟動影片分割 GUI
python tool/video_splitter_gui.py

# 執行影片同步截圖
python tool/video_grab_photo.py
```

## 📦 依賴套件

使用這些工具前，請確保已安裝以下依賴：
- `opencv-python`
- `pymediainfo`
- `tqdm`
- `tkinter` (通常隨 Python 安裝)
- `ffmpeg` (需安裝於系統路徑中)
